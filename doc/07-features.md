# 07 · 功能清单与验收标准

> 验收以 **GIVEN-WHEN-THEN** 表达。每个功能给出最小可发布定义。

## F1 项目管理（多作品）

- 列出 / 新建 / 重命名 / 归档 / 删除作品
- 启动时记忆上次打开的作品

**AC**
- _GIVEN_ 用户首次启动 _WHEN_ 进入应用 _THEN_ 显示 Welcome 页，引导新建作品。
- _GIVEN_ 已存在 N 个作品 _WHEN_ 启动 _THEN_ 自动打开 `last_project_id`，不存在则回到 Welcome。
- 删除作品需二次确认；级联删除其下所有章节/角色/大纲。

## F2 章节管理

- 树形结构（卷 → 章 → 节）三级
- 拖拽排序、键盘上下移动
- 章节状态：草稿 / 修订中 / 已完成（图标提示）
- 右键菜单：新建子章节 / 重命名 / 复制 / 删除 / 导出

**AC**
- 拖拽 100 个节点的树性能 < 16ms 帧时长。
- 删除带子节点章节需二次确认，且记入最近 30 天可恢复（通过 `revisions`）。

## F3 角色管理

- 角色卡：姓名、别名、头像、身份、画像（Markdown）、自定义属性
- 关系：成对添加（父子、师徒等）
- 列表 / 卡片视图切换

**AC**
- 角色卡支持引用其他角色（`@姓名`）自动跳转。
- 一键"插入角色卡到当前段落"。

## F4 大纲管理

- 树形（自由层级）
- 节点 ↔ 章节双向关联（一个大纲点可绑定到一个章节）
- 状态：构思 / 撰写中 / 已完成

**AC**
- 大纲拖拽到章节树上自动创建对应章节。
- 章节标题改名时大纲绑定保持不变。

## F5 编辑器

- TipTap 内核
- 富文本：标题 H1-H3、加粗、斜体、下划线、删除线、引用、列表（有序/无序/任务）、分隔线、代码块
- 节点扩展：场景分割（特殊样式）、注释 mark、TODO mark
- 工具栏 + 浮动菜单
- 自动保存（防抖 500ms）
- 历史撤销/重做（无限）
- 中文输入法兼容

**AC**
- 输入 1 万字稳定流畅（无明显卡顿）。
- 关闭应用 → 重新打开，最近一次输入完整呈现。

## F6 段落精修（AI）

- 选中段落 → 弹 popover
- 风格：凝练 / 扩写 / 换语气（古风/现代/口语等） / 翻译 / 自由提示
- 流式输出 + diff 对照
- "应用替换" 或 "插入下方"

**AC**
- 选中范围超 5000 字时给出"段落较长，建议分段"提示。
- 网络中断时显示已生成部分，提供"继续生成"。

## F7 章节校审（AI）

- 当前章节做四维（逻辑/连贯/人物口吻/伏笔）评审
- 输出 JSON → 渲染为可点击建议列表
- 接受 / 忽略每条建议

**AC**
- 报告生成后 `ai_messages.scope = 'review'` 留痕。
- 每条建议 "跳转" 到对应行，定位偏差 < 1 行。

## F8 字数与统计

- 实时字数：本段 / 本章 / 全书
- 中文按字符（不计空格/标点可切换）
- 写作会话自动开始：编辑器获取焦点时；30 秒无输入自动结束
- 每日字数累计写入 `writing_sessions`

**AC**
- Dashboard 显示：今日/本周/本月字数、连续写作天数（streak）、目标进度、30/90/365 天热力图。
- 切换作品后看板数据按 project 隔离。

## F9 批注 / 标记

- 选中文字 → 添加 highlight / comment / TODO
- 右侧批注面板汇总当前章节所有批注
- 点击跳转至原位置
- 完成 TODO 一键解决

**AC**
- 同一段同时可叠加多种标记。
- 删除被标记区域时自动清理对应 annotation 记录。

## F10 全文检索

- Ctrl/Cmd+Shift+F 唤起全局搜索
- 搜索范围：章节标题 / 章节正文 / 角色 / 大纲
- 结果按作品 / 类别分组
- 关键词高亮 + 前后 30 字 snippet

**AC**
- 50 万字稿件查询 < 200ms（FTS5）。

## F11 命令面板

- Ctrl/Cmd+K
- 模糊搜索：章节、角色、命令、设置项
- 最近使用置顶

## F12 设置

- 主题（亮/暗/护眼）
- 字体（族 + 大小 + 行高）
- AI provider 配置（密钥、base_url、模型）
- 备份目录、是否自动备份
- 中英文计数模式

## F13 导出

- Markdown（每章一个 .md / 单文件合并）
- Docx（pandoc 或 `docx-rs`）
- Plain Text

**AC**
- 50 章作品 5 秒内完成导出。
- 元数据（作者 / 项目名 / 字数）写入 docx 属性。

## F14 自动备份

- 默认每天 03:00 复制 db 到用户配置目录
- 保留最近 7 天，超出删除最旧

## F15 跨平台与 CI

- macOS .dmg、Windows .msi 通过 GitHub Actions 自动产出
- 打 tag `v*` 触发发布
