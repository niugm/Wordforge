# 01 · 产品概览

## 1.1 一句话定义

**Wordforge** 是一款本地优先的桌面写作工具，把编辑器、章节/角色/大纲管理、写作统计、AI 协作整合在同一个轻量应用内，为长文创作者提供"从构思到定稿"的完整工坊。

## 1.2 目标用户

| 画像 | 痛点 | 我们提供 |
|---|---|---|
| 网络小说作者 | 章节多，前后线索断、人物口吻飘 | 章节树 + 角色卡 + AI 校审 |
| 剧本/编剧 | 大纲反复迭代、需场景级标注 | 大纲树 + 行内批注 + 修订历史 |
| 专栏/非虚构作者 | 段落需反复打磨 | AI 段落精修（凝练/扩写/换语气） |
| 多语种作者 | 需要快速切换风格、翻译 | 内置翻译/改写工作流 |

## 1.3 设计哲学

1. **本地优先 (Local-first)**：稿件、密钥、统计全部存在用户本机；离线可用。
2. **简洁 (Minimal)**：默认隐藏复杂功能，专注于"写"这件事。
3. **可扩展 (Composable)**：编辑器/AI provider 都是可插拔模块。
4. **隐私 (Private)**：API key 加密存储，AI 调用从 Rust 侧发起，前端不接触原文 key。
5. **键盘友好 (Keyboard-first)**：所有高频操作有快捷键 + 命令面板。

## 1.4 用户故事 (User Stories)

> **US-1** _作为一个小说作者，我希望_ 在创建项目后看到清晰的章节树，_以便_ 用拖拽组织章节顺序。
>
> **US-2** _作为一个长文作者，我希望_ 选中段落后调出"AI 精修"，挑选风格后看到对比 diff，_以便_ 决定是否替换原文。
>
> **US-3** _作为一个剧本作者，我希望_ 给某段对白打上 TODO 标记，_以便_ 之后从右侧批注面板一键跳回。
>
> **US-4** _作为一个日更作者，我希望_ 看到今日字数 / 连续写作天数 / 月度热力图，_以便_ 维持创作节奏。
>
> **US-5** _作为一个隐私敏感的作者，我希望_ 我的 OpenAI key 永远不出现在任何明文文件中，_以便_ 即使设备被他人短暂使用也安全。
>
> **US-6** _作为一个跨设备作者，我希望_ 一键备份数据库到指定目录，_以便_ 自己用网盘或 Git 同步。

## 1.5 范围 (In / Out of Scope)

### v0.1.0 In

- 桌面 App（macOS / Windows）
- 多作品（Project）切换
- 章节 / 角色 / 大纲 / 编辑器 / 批注
- 字数统计 + 写作看板
- AI：OpenAI 兼容 / Anthropic / Gemini
- 本地 SQLite + 全文检索
- 一键导出 Markdown / Docx / Txt
- GitHub Actions 双平台打包

### v0.1.0 Out（后续版本）

- 云同步（Dropbox/iCloud/自建 WebDAV）
- 协同编辑
- Linux 包（理论可行，留作社区贡献）
- 移动端
- 插件市场
