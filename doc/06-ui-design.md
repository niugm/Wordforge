# 06 · UI 设计

## 6.1 设计原则

1. **写作优先**：编辑器永远占据视觉中心。
2. **侧栏可折叠**：左右两栏都能一键收起，进入"专注模式"。
3. **暗色主题为默认**（深夜创作友好），同时提供亮色 / 护眼。
4. **最小动效**：仅过渡 / 提示，不做装饰性动画，避免分心。
5. **键盘第一**：所有动作都有快捷键，命令面板 (Ctrl/Cmd+K) 可达。

## 6.2 信息架构

```
Wordforge
├─ Welcome / Project Picker        （首次启动 / 切换作品）
└─ Workspace                       （主写作界面）
   ├─ Left Sidebar
   │   ├─ Tab: 章节树
   │   ├─ Tab: 角色
   │   ├─ Tab: 大纲
   │   └─ 底部：项目切换 / 设置
   ├─ Center: 编辑器
   │   ├─ TitleBar（章节名 + 状态 + 字数）
   │   ├─ TipTap
   │   └─ Footer（保存状态 / 字数统计 / 焦点模式按钮）
   └─ Right Sidebar
       ├─ Tab: AI 助手
       ├─ Tab: 批注
       └─ Tab: 修订历史

Modals / Drawers
├─ Command Palette (Ctrl/Cmd+K)
├─ Settings (Ctrl/Cmd+,)
├─ Search (Ctrl/Cmd+Shift+F, 全文检索)
└─ Polish Popover (选中段落时弹出)

Standalone Pages
├─ Dashboard (Ctrl/Cmd+D)
└─ Export Wizard
```

## 6.4 主题与排版

| 模式 | 背景 | 前景 | 强调 |
|---|---|---|---|
| 暗色（默认） | `#0F1115` | `#E5E7EB` | `#A78BFA` |
| 亮色 | `#FAFAFA` | `#1F2937` | `#7C3AED` |
| 护眼 | `#F0E7D8`(米黄) | `#3E372C` | `#8B5E3C` |

字体：
- 中文：`Noto Sans SC`、`PingFang SC`、`Microsoft YaHei` fallback
- 西文：`Inter`、`-apple-system`
- 编辑器正文：可切换 衬线（`Source Han Serif`）/ 无衬线
- 行高：编辑器 `1.85`（长文阅读舒适）

## 6.5 快捷键 (默认)

## 6.6 可达性 (a11y)

- 所有交互组件来自 Radix Primitives，键盘导航 + ARIA 默认就位
- 颜色对比度满足 WCAG AA
- 编辑器支持系统级缩放（Ctrl+= / Ctrl+-）
